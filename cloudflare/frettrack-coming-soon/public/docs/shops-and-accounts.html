Shops and accounts

Manage shop access

A FretTrack account belongs to a user. A shop contains the repair records, staff roles, plan status, inventory, jobs, customers, and settings for that repair business.

Account basics

  • Use the approved beta email address when signing in.
  • Verify your email if the sign-in flow asks you to do so.
  • After approval, create or select the shop you are testing or operating.
  • Use one account per person so work logs and activity can be tied to the right user.

Shop settings

Shop settings may include shop profile details, team members, inventory locations and categories, plan or trial status, and other operational preferences. Owners and admins usually manage these settings.

Trial, Shop, and Pro

Trial is temporary evaluation access. Shop is the paid core repair workflow. Pro adds upgraded tools such as advanced reports, team members, and photo editing where available.

If a trial expires, shop data is preserved. Some write actions or paid features may be unavailable until access is restored.

Team members

Team member access should match each person's job. Owners manage the account, admins help run operations, techs handle repair workflow, and viewers can review records without editing them.

If something looks wrong

If you can see a shop you should not access, or if support-only tools appear for a normal shop user, report it immediately through support.
